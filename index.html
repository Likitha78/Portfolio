My Projects

Portfolio

One notable project was the creation of a portfolio using HTML and CSS, where I aimed to showcase my projects and certifications. To achieve this, I designed a user-friendly and visually appealing website. I implemented various web design principles and utilized CSS for styling, resulting in a professional portfolio that effectively highlighted my accomplishments.

Snakes and Ladders

I developed a Snakes and Ladders game using Python. The goal was to create an engaging and functional game that simulated the classic board game. I wrote the entire game logic in Python, designed the user interface, and tested the game to ensure it was bug-free. The final product was a fully functional digital version of Snakes and Ladders that provided a seamless gaming experience.

Tic Tac Toe

I also worked on a Tic Tac Toe game using C, where the task was to develop a simple yet entertaining game. I focused on writing efficient code for game logic and implemented a basic graphical user interface. The game allowed two players to compete against each other, and I ensured that it was both fun and user-friendly.

Calculator

I worked on a project to develop a calculator using Java, with the objective of creating a functional and user-friendly application for performing basic arithmetic operations. The task was to design a calculator that could handle addition, subtraction, multiplication, and division, while providing a simple and intuitive user interface.

Smart Dustbin Level Indicator

I created a smart dustbin level indicator using Tinkercad to address waste management issues. The task was to design a system that could monitor and report the fill level of a dustbin. I used Tinkercad for circuit design and simulation, and programmed sensors to detect the level of waste. The final prototype effectively indicated when the dustbin needed to be emptied.

Smart Door Lock System

I developed a smart door lock system using ESP8266. The objective was to create a secure and remotely controllable door lock. I programmed the ESP8266 module to communicate with a mobile app for remote access and control. The system included security features such as password protection and real-time alerts, resulting in a practical and secure smart door lock solution.

Automatic Speed Control System

I implemented an automatic speed control system using Arduino Uno. The objective was to design a system that could control the speed of a vehicle automatically based on road conditions. I programmed the Arduino to process sensor inputs and adjust the speed accordingly. This project involved hardware integration and coding, resulting in a functional prototype that demonstrated the feasibility of automatic speed control.

Wireless Mobile Charging

Wireless Mobile Charging project, where I focused on the practical implementation of mutual induction and an AC to DC converter. I researched the principles of wireless charging, designed the circuitry, and conducted tests to ensure efficient power transfer. The outcome was a working prototype that successfully charged mobile devices wirelessly.

Traffic Light System

I also developed a traffic light system using KEIL and Proteus. The task was to create a simulation of traffic lights using microcontroller programming. I wrote the code in KEIL, designed the circuit in Proteus, and tested the system to ensure accurate light sequences. The project demonstrated my ability to work with microcontroller simulations and programming.

Optimization of CIGS Solar Cells

For a research-oriented project, I worked on the optimization of CIGS solar cells with different back contacts using SCAPS-1D, which was published in the Journal of Optics. The goal was to improve the efficiency of solar cells by experimenting with various back contact materials. I used SCAPS-1D simulation software to model the solar cells and analyze the results. The research provided valuable insights into the optimization of solar cell performance.
